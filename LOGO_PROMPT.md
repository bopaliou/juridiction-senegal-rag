# Prompt pour génération du logo YoonAssist AI

## Prompt principal pour Nano Banana

```
Créer un logo moderne et professionnel pour "YoonAssist AI", un assistant juridique intelligent pour le droit sénégalais.

ÉLÉMENTS VISUELS REQUIS :
- Une balance de la justice (symbole juridique classique) stylisée et moderne
- Des éléments d'intelligence artificielle : des particules lumineuses ou des éclats autour de la balance
- Une référence subtile au Sénégal : peut-être une étoile verte (symbole du drapeau sénégalais) ou des couleurs du drapeau (vert, jaune, rouge) intégrées de manière élégante
- Le texte "YoonAssist" en typographie moderne et lisible
- Un badge ou indicateur "AI" discret

STYLE ET ESTHÉTIQUE :
- Design moderne, épuré et professionnel
- Style flat design ou minimaliste avec des gradients subtils
- Couleurs principales : bleu (#2563eb, #3b82f6) pour la confiance et le professionnalisme
- Couleurs secondaires : vert émeraude (#10b981) pour l'intelligence, indigo (#6366f1) pour la profondeur
- Accents : or/jaune (#fbbf24) pour l'excellence, peut-être pour l'étoile sénégalaise
- Typographie : police moderne, sans-serif, bold pour "YoonAssist", plus petite pour "AI"

COMPOSITION :
- La balance de la justice au centre, légèrement stylisée avec des lignes modernes
- Des particules lumineuses ou des éclats (sparkles) autour pour représenter l'IA
- Le texte "YoonAssist" en dessous ou intégré de manière harmonieuse
- Un petit badge "AI" en haut à droite ou intégré dans le design
- Éventuellement une étoile verte discrète pour la référence sénégalaise

FORMAT ET UTILISATION :
- Logo principal : format carré ou rectangulaire horizontal
- Version icône : version simplifiée sans texte pour les favicons
- Fond transparent ou blanc selon le contexte
- Adaptable pour usage sur fond clair et foncé

CONTEXTE ET SIGNIFICATION :
- YoonAssist = Assistant (Yoon) + Assistance
- Domaine : Droit sénégalais (Constitution, Code du Travail, Code Pénal, etc.)
- Slogan : "L'expertise juridique sénégalaise à votre service"
- Public : Professionnels du droit, citoyens cherchant des informations juridiques
- Valeurs : Expertise, accessibilité, modernité, confiance, intelligence

RÉSULTAT ATTENDU :
Un logo qui évoque à la fois la tradition juridique (balance) et la modernité technologique (IA), avec une touche sénégalaise subtile, dans un style professionnel et élégant qui inspire confiance et expertise.
```

## Variantes du prompt

### Version courte (pour tests rapides)
```
Logo moderne pour "YoonAssist AI" : balance de la justice stylisée avec particules lumineuses (IA), couleurs bleu (#3b82f6) et vert émeraude (#10b981), typographie moderne, style professionnel et épuré, référence subtile au Sénégal (étoile verte), badge "AI" discret.
```

### Version détaillée avec spécifications techniques
```
Créer un logo vectoriel pour "YoonAssist AI" avec les spécifications suivantes :

COMPOSANTS :
1. Balance de la justice : 
   - Design moderne et minimaliste
   - Lignes épurées, pas trop détaillées
   - Position : centre ou légèrement à gauche
   - Couleur principale : bleu (#2563eb)

2. Éléments IA :
   - 3-5 particules lumineuses (sparkles) autour de la balance
   - Couleur : vert émeraude (#10b981) ou bleu clair
   - Taille variable pour dynamisme

3. Étoile sénégalaise :
   - Petite étoile verte à 5 branches
   - Position : discrète, peut-être intégrée dans la balance ou à côté
   - Couleur : vert (#059669)

4. Texte "YoonAssist" :
   - Police : Inter, Poppins, ou Geist (moderne, sans-serif)
   - Poids : Bold (700)
   - Taille : proportionnelle au logo
   - Couleur : bleu foncé (#1e40af) ou noir (#0f172a)

5. Badge "AI" :
   - Petit badge arrondi
   - Fond : bleu clair (#dbeafe)
   - Texte : bleu foncé (#1e40af)
   - Position : coin supérieur droit

PALETTE DE COULEURS :
- Bleu principal : #2563eb (blue-600)
- Bleu foncé : #1e40af (blue-800)
- Vert émeraude : #10b981 (emerald-500)
- Vert foncé : #059669 (emerald-600)
- Indigo : #6366f1 (indigo-500)
- Or/Jaune : #fbbf24 (amber-400) - optionnel pour étoile

STYLE :
- Flat design avec gradients subtils
- Ombres douces pour profondeur
- Lignes nettes et précises
- Équilibre entre tradition (balance) et modernité (IA)

FORMATS :
- Logo complet : 800x200px (horizontal) ou 400x400px (carré)
- Icône : 256x256px (carré, sans texte)
- Favicon : 32x32px, 64x64px
- Fond transparent (PNG) et versions avec fond blanc
```

## Instructions pour Nano Banana

1. **Copier le prompt principal** dans Nano Banana
2. **Ajuster selon les préférences** :
   - Si vous préférez plus de modernité : accentuer les particules lumineuses
   - Si vous préférez plus de tradition : accentuer la balance
   - Si vous voulez plus de référence sénégalaise : rendre l'étoile plus visible
3. **Générer plusieurs variantes** pour choisir la meilleure
4. **Tester sur différents fonds** (clair et foncé)

## Notes de design

- **Équilibre** : Le logo doit équilibrer tradition juridique et innovation technologique
- **Lisibilité** : Le texte "YoonAssist" doit être clairement lisible même en petite taille
- **Scalabilité** : Le logo doit fonctionner de 32px (favicon) à 800px (header)
- **Cohérence** : Les couleurs doivent correspondre à celles utilisées dans l'interface (bleu, vert émeraude)
- **Professionnalisme** : Le design doit inspirer confiance et expertise juridique
